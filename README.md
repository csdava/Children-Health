# 童视智圈
## 项目简介
基于边缘视觉AI打造的家校共育儿童健康管理平台，搭建学校、家长、儿童三方协同使用体系。依托轻量化YOLO26模型完成餐食图像识别，智能分析儿童膳食营养结构；接入小米手环终端设备，实时采集心率、运动量等生理健康数据。平台打通多端数据链路，实现饮食评估、健康监测、家校信息互通，全方位守护儿童身心健康成长。

## 技术架构
- 核心模型：YOLO26 轻量化视觉检测模型
- 开发语言：Python
- 核心技术：边缘端AI推理、智能图像识别、智能穿戴设备数据对接
- 业务能力：多用户权限管理、三端数据同步、健康数据分析统计

## 核心功能
1. 膳食智能识别：通过YOLO26模型识别餐食种类，自动评估饮食营养配比
2. 健康数据采集：对接小米手环，同步收录心率、日常活动量等体征数据
3. 三方协同管理：划分儿童、家长、学校独立权限，实现信息互通查阅
4. 健康状态分析：整合饮食与身体数据，生成儿童健康综合评估报告
5. 家校联动提醒：异常健康状态及时预警，方便家校共同监护管理
